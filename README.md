# tel-blacklist-react

## 功能描述 (Features)
列出電話名單，使用者可以查閱、增加、修改、刪除一筆電話。
也能夠使用電話或姓名搜尋。

## 安裝與執行步驟 (Installation and Execution)
1. 將專案複製到本機 (兩種方法)
> (1) 打開終端機輸入 
`git clone https://github.com/aoigj100a/order-comm-react`</br>
> (2) 點選 download ZIP 下載

2. 進入專案資料夾安裝工具包
> 打開終端機輸入
`npm install`

3. 執行專案
> 打開終端機輸入 
`npm start`

4. 使用瀏覽器瀏覽
> 打開瀏覽器在網址列輸入 localhost:3000 即可瀏覽
